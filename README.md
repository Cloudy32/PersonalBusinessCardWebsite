# RB.code — personal business card website  

# Сам сайт: [клик](https://rbcoder.ru/)

`RB.code` — личный сайт-визитка Бондаря Романа Михайловича, Python Backend Developer. Проект сделан как аккуратная frontend-страница для знакомства со мной, моими проектами, услугами и контактами.

Сайт собран в формате одностраничного портфолио: первый экран с приветствием, блок «Обо мне», раздел проектов, список услуг и ссылки на социальные сети.

## Что есть на сайте

- Первый экран с названием `RB.code` и приветственным текстом.
- Блок «Обо мне» с фотографиями, краткой историей пути в IT и текущим стеком.
- Раздел проектов с карточками и ссылками на GitHub.
- Раздел услуг с описанием направлений работы и вариантами стоимости.
- Переключение языка RU / ENG.
- Ссылки на GitHub, Telegram, Instagram и email.
- Анимации появления текста и блоков при скролле.
- Адаптивная вёрстка для разных размеров экрана.

## Проекты в портфолио

На сайте представлены мои pet-проекты и учебно-практические backend-работы:

- **Looking For Restaurants (LFRest)** — сервис для поиска ресторанов по предпочтениям пользователя с ИИ-ассистентом.
- **News Bot** — Telegram-бот для автоматической подборки новостей из RSS-источников.
- **Linkcuts** — сервис для сокращения длинных ссылок.
- **My Neighbor** — Telegram-бот для поиска сожителей и людей для совместной аренды жилья.
- **Payment Service** — backend-сервис для асинхронной обработки платежей с очередью, outbox-подходом, webhook, retry и DLQ.

## Локальный запуск

Самый простой способ открыть сайт локально:

```bash
cd app
python -m http.server 8001 --bind 127.0.0.1
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:8001/index.html
```

Если нужно установить Python-зависимости из `requirements.txt`:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```
